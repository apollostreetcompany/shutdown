# IRS Business Closure Workflow

## Purpose
Guide a user through closing their federal tax accounts with the IRS after state dissolution.

## When to Use
After state dissolution is filed and confirmed.

## Workflow

### Step 1: File Final Federal Tax Return
1. **For LLC (single-member)**: File final Schedule C with personal Form 1040
2. **For LLC (multi-member)**: File final Form 1065 (Partnership Return)
3. **For S-Corp**: File final Form 1120-S
4. **For C-Corp**: File final Form 1120
5. Check the "Final return" box on the top of the form
6. Mark the final period covered

### Step 2: File Final Employment Tax Returns (if had employees)
1. File final Form 941 (Quarterly)
2. File final Form 940 (Annual FUTA)
3. Check the "Final" box on each form
4. Issue final W-2s to all employees by January 31
5. File W-3 transmittal

### Step 3: Close EIN
1. Write a letter to IRS including:
   - Complete legal name of the entity
   - EIN
   - Business address
   - Reason for closing (dissolution)
   - Date of dissolution
2. Mail to: Internal Revenue Service, Cincinnati, OH 45999
3. Or fax to: (855) 641-6935

### Step 4: Cancel Remaining Tax Accounts
1. If had excise tax obligations: File final Form 720
2. If sold alcohol/tobacco/firearms: Notify TTB
3. If had fuel tax obligations: File final Form 720

### Step 5: Record Keeping
- Keep all tax records for at least 7 years after the final return
- Keep employment tax records for at least 4 years after the tax is due or paid
- Keep records of any assets distributed to owners
